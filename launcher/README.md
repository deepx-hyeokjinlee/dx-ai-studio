# Launcher — DX AI Studio Hub

The home screen of DX AI Studio and the single entry point you open in the browser. It
starts every module for you and lets you jump into any of them from one place.

## Using it

1. Launch the studio (`./launcher.sh`) and open the shown address in your browser.
2. Wait for the boot screen — it starts all modules; a status dot per module turns green
   when ready.
3. From the home hub, click a module to open it:
   **DX App**, **DX Stream**, **DX Model Zoo**, **DX Compiler**, **DX EdgeGuide**,
   **DX Benchmark**, **DX Monitor**, **DX Agent Dev**.
4. Also on the hub:
   - **SDK Library** — browse the DEEPX SDK documentation and brochures in-app.
   - **About DEEPX** — company, product, and news information.
   - **Language** toggle (top-right) — 6 languages.
   - **Buy now** banner — jump to the DEEPX store.

## Key features

- **One-click access** to all eight modules from a single hub.
- **SDK Library** — curated DEEPX docs/PDFs viewable without leaving the studio.
- **About DEEPX** — product and news overview.
- **Purchase entry point** — a dismissible "Buy now" banner linking to the DEEPX store.
- **6-language UI** (English / 한국어 / 日本語 / 简体中文 / 繁體中文 / Español).
